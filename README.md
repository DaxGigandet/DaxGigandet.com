This repository hosts the **source code** for the **official website** of Dax Gigandet. It is designed to be a **fast, modern, and easy-to-navigate** website, featuring:  

- **Official Social Media Links** 📲  
- **Latest Music Releases** 🎵  
- **Press & Media Mentions** 📰  
- **Advocacy & Activism** 🏳️‍🌈  
- **Blog & Personal Updates** ✍️

The site is built using **GitHub Pages** and **Jekyll**, making it a **lightweight, static, and fast-loading** website.

## 🛠️ Installation & Local Development  

### 🔹 Run Locally  
Clone the repository and install Jekyll to preview the site on your computer:  

```bash
git clone https://github.com/DaxGigandet/DaxGigandet.com.git
cd DaxGigandet.com
bundle install
bundle exec jekyll serve
